# Age Calculator

A professional, SEO-optimized age calculator web application that calculates exact age in years, months, days, hours, minutes, and seconds.

## Features

- ✅ Calculate exact age from birth date to any target date
- ✅ Shows age breakdown in multiple units (years, months, weeks, days, hours, minutes, seconds)
- ✅ Next birthday countdown
- ✅ Fully responsive design
- ✅ SEO optimized with meta tags and schema markup
- ✅ Clean, modern UI
- ✅ AdSense ready

## How to Deploy to GitHub Pages

### Step 1: Create a GitHub Account
1. Go to https://github.com
2. Click "Sign up" and create a free account

### Step 2: Create a New Repository
1. Click the "+" icon in the top right corner
2. Select "New repository"
3. Repository name: `age-calculator` (or any name you prefer)
4. Make it **Public**
5. ✅ Check "Add a README file"
6. Click "Create repository"

### Step 3: Upload Your Files
1. In your repository, click "Add file" → "Upload files"
2. Drag and drop these files:
   - `index.html`
   - `style.css`
   - `script.js`
3. Scroll down and click "Commit changes"

### Step 4: Enable GitHub Pages
1. In your repository, click "Settings"
2. Click "Pages" in the left sidebar
3. Under "Source", select "main" branch
4. Click "Save"
5. Wait 1-2 minutes for deployment

### Step 5: Access Your Website
Your site will be live at:
```
https://yourusername.github.io/age-calculator/
```

Replace `yourusername` with your actual GitHub username.

## Customization

### Update URLs in index.html
Open `index.html` and replace `yourusername` with your actual GitHub username in these lines:
- Line 11: `<link rel="canonical" href="https://yourusername.github.io/age-calculator/">`
- Line 14-15: Open Graph URLs
- Line 19-20: Twitter URLs
- Line 26: Schema.org URL

### Add Google AdSense
1. Apply for Google AdSense at https://www.google.com/adsense
2. Once approved, replace the ad placeholder in `index.html` (around line 112) with your AdSense code

### Add Analytics
Add Google Analytics code before `</head>` tag in `index.html` to track visitors.
